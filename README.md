# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
Play against another player in the browser with a clean UI and smooth gameplay.

---

## 🚀 Features

- 🎯 Two player gameplay
- ✨ Hover effects on cells
- 🧠 Win detection logic
- 🔄 Restart / Reset game
- 🎨 Responsive UI
- ⚡ Fast and lightweight
